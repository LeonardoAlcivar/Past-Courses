Computes the time measured by the Timer class after a complete start/stop cycle.


Utilizes stored beginTime and duration values to return an elapsed time in seconds. This function will throw a logic error if an elapsed time is solicited, but the timer is still being run (that is, the timer has not been stopped).

Starts the timer for the Timer class.


Uses "wall clock" functionality to get a beginning time. This time is then stored in the beginTime data member. Will throw a runtime error if the gettimeofday function fails.

          )

Stops the timer in the Timer class.


Uses the "wall clock" functionality to record a stopping time in the duration data member. This time will be compared to the timer's start time to compute and elapsed time. Will throw a logic error if the the timer has not been started. Throws a runtime error if the gettimeofday function fails.
