Detailed Description

Contains definitions for cunctions that together comprise a heap sort utility. The heapSort() shell function is provided by the lab manual package, while the moveDown helper function was written by T. Henriod. This heapSort is for arrays.

A shell program that utilizes the PriorityQueue ADT to simulatean operating system's use of a priority queue to regulate access to a system resource (printer, disk, etc.).

The driving function of the program. A number of differing priority levels and a length of time to run the simulation are prompted for, and then the simulation is run, randomly adding 0, 1, or 2 tasks of random priority each time. Each time a task is "processed," its summary statistics are reported.
