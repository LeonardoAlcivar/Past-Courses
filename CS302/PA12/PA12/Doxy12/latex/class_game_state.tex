\subsection{\-Detailed \-Description}
\-Contains the full amount of data required to represent and manipulate a \-Rush \-Hour \-Game state. \-Includes a representation of a game board, the ability to move cars, and overloaded copy constructor and assignment operator for use with the \-S\-T\-L queue.
