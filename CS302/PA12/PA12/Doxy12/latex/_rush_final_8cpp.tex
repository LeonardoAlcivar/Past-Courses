solve\-Rush\-Hour

\-Finds the lowest number of moves required to find the solution to a \-Rush \-Hour puzzle. \-If a solution cannot be found, an error code is returned.


\begin{DoxyParams}{\-Parameters}
{\em num\-Cars} & \-The number of cars to be used in a given puzzle. \\
\hline
{\em initial} & \-The initial board for a given problem. \\
\hline
{\em states\-To\-Evaluate} & \-A queue that will be used in a breadth first search to contain every game state that will be considered. \\
\hline
{\em observed\-States} & \-A map used to contain an identifying key string that contains vehicle list data used to check to see if a state has already been considered.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{\-Returns}
num\-Moves \-The minimum number of moves required to solve the given puzzle.
\end{DoxyReturn}
\begin{DoxyPrecond}{\-Precondition}

\begin{DoxyEnumerate}
\item \-A valid number of cars that corresponds precisely to the data waiting to be read in is required.
\item states\-To\-Evaluate must be an empty queue.
\item observed\-States must be empty as well.
\item \-The initial state will be representative of the starting condition for the puzzle to be solved.
\end{DoxyEnumerate}
\end{DoxyPrecond}
\begin{DoxyPostcond}{\-Postcondition}

\begin{DoxyEnumerate}
\item \-A number of moves corresponding to either the number of moves needed to solve a given puzzle or an error code for an unsolvable puzzle is returned.
\end{DoxyEnumerate}
\end{DoxyPostcond}
{\bfseries } 
\begin{DoxyEnumerate}
\item \-A breadth first search is implemented.
\item \-Game\-States are added to the queue and their representative strings are added to the queue if they are newly encountered.
\item \-States are iteratively popped off the queue and checked to see if they are a winning state.
\item \-If they are not, all possible \-Game\-States based on legal moves are generated and stored in the queue. \-Their representative strings and the number of moves used up to that point are stored for future reference.
\item \-Should the queue ever become empty, this indicates that all possible moves have been attempted and the puzzle is unsolvable.
\end{DoxyEnumerate}
